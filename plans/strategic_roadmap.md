# Ante Apocalypse 전략적 로드맵

## 프로젝트 개요

**프로젝트명:** Ante Apocalypse  
**장르:** 서바이벌 호라 FPS / 로그라이크  
**플랫폼:** WebGL (Babylon.js) → 향후 멀티플랫폼 확장  
**타겟:** 코어/하드코어 게이머, 로그라이크 팬

---

## 1. 프로젝트 비전 및 목표

### 1.1 비전 선언문

> "웹 브라우저에서 AAA급 FPS 경험을 제공하며, 로그라이크 요소와 결합하여 매번 새로운 도전과 전략적 깊이를 선사하는 서바이벌 호라 게임"

### 1.2 핵심 목표

| 목표 유형           | 설명                                 | 우선순위 |
| ------------------- | ------------------------------------ | -------- |
| **기술적 목표**     | WebGL 기반 고성능 3D FPS 엔진 구축   | P0       |
| **게임플레이 목표** | 직관적이면서도 전략적인 전투 시스템  | P0       |
| **네트워크 목표**   | 안정적인 실시간 멀티플레이어 (Co-op) | P1       |
| **비즈니스 목표**   | 지속 가능한 콘텐츠 업데이트 모델     | P1       |
| **확장성 목표**     | 모바일/콘솔 포팅 가능한 아키텍처     | P2       |

### 1.3 현재 기술 스택 분석

```
┌─────────────────────────────────────────────────────────────┐
│                    기술 스택 현황                            │
├─────────────────────────────────────────────────────────────┤
│  엔진: Babylon.js 8.47.0 (WebGL)                            │
│  언어: TypeScript 5.7.3                                      │
│  빌드: Vite 6.0.7                                            │
│  패키지: pnpm 워크스페이스 모노레포                          │
│  네트워크: Photon Realtime 4.4.0                             │
│  상태관리: Nanostores 1.1.0                                  │
│  테스트: Vitest 4.0.18                                       │
│  인프라: Firebase (Hosting, Firestore)                       │
└─────────────────────────────────────────────────────────────┘
```

### 1.4 아키텍처 강점

- **Controller-Pawn 패턴**: Unreal Engine 스타일의 명확한 관심사 분리
- **Data-Driven Design**: JSON 기반 무기/적 데이터 관리
- **Component 기반**: 확장 가능한 ECS-like 구조
- **Event-Driven**: Type-safe Observable 패턴
- **하이브리드 네트워크**: Photon + Local Server 지원
